# 巡花问柳大世界

这是一个静态网站，不需要安装依赖。
把整个项目文件夹放到哪里都可以，直接打开 `index.html` 就能本地查看。

如果你以后要自己更新内容，先看这两份说明：

- [更新手册.md](./更新手册.md)
- [一键更新说明.md](./一键更新说明.md)

## 以后主要改哪里

1. `data/site-data.js`
2. `assets/trips/`
3. `assets/cats/`
4. `assets/shared/`

## 一键发布

你改完内容后，直接双击：

`一键更新网站.bat`

它会自动把改动提交并推送到 GitHub，随后 GitHub Pages 会自动更新网站。

## 搬家注意

这个项目现在已经是可搬家的版本，但要注意两点：

1. 要移动整个文件夹，不要只移动一部分文件
2. `.git` 文件夹必须保留，不然就不能继续一键发布
