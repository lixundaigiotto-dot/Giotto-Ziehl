# 一键更新说明

以后你更新完网站内容后，直接双击：

`一键更新网站.bat`

它会自动帮你完成这几步：

1. 检查有没有新的改动
2. 自动执行 `git add .`
3. 自动执行 `git commit`
4. 自动执行 `git push origin main`
5. 触发 GitHub Pages 自动重新发布

也就是说：

**你改完内容后，点一次它，基本就会自动发布并更新网站。**

但它只负责“上传和发布”，不会自动帮你写内容。

你每次还是要先自己改好这些东西，比如：

- 把新照片放进 `assets/trips/`
- 把新视频放进 `assets/trips/`
- 修改 `data/site-data.js`
- 本地打开 `index.html` 看看页面对不对

然后再双击 `一键更新网站.bat`。

## 现在为什么更适合搬家了

这套脚本现在优先使用你电脑里已经安装好的 `git`。
如果当前机器没有系统 Git，它才会再尝试备用路径。

所以只要你把整个项目文件夹一起搬走，并且保留 `.git`，通常还是可以继续发布。

如果脚本成功运行，通常等 1 到 3 分钟，刷新这个网址就能看到更新：

`https://lixundaigiotto-dot.github.io/Giotto-Ziehl/`
